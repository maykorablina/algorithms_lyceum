## Bubble Sort (–°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –ø—É–∑—ã—Ä—å–∫–æ–º)

–ï—Å–ª–∏ –ª–µ–≤—ã–π —ç–ª–µ–º–µ–Ω—Ç –±–æ–ª—å—à–µ –ø—Ä–∞–≤–æ–≥–æ, —Ç–æ –º—ã **–º–µ–Ω—è–µ–º –∏—Ö –º–µ—Å—Ç–∞–º–∏**: `5, 4` ‚Üí `4, 5`

–¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, –±–æ–ª—å—à–∏–µ —ç–ª–µ–º–µ–Ω—Ç—ã ¬´–≤—Å–ø–ª—ã–≤–∞—é—Ç¬ª –≤ –∫–æ–Ω–µ—Ü –º–∞—Å—Å–∏–≤–∞, –∫–∞–∫ –ø—É–∑—ã—Ä–∏ ‚Äî –æ—Ç—Å—é–¥–∞ –∏ –Ω–∞–∑–≤–∞–Ω–∏–µ.

### –ü—Ä–∏–º–µ—Ä

1.  –°—Ä–∞–≤–Ω–∏–≤–∞–µ–º 5 –∏ 3 ‚Üí –º–µ–Ω—è–µ–º ‚Üí `[3, 5, 4, 1]`

3.  –°—Ä–∞–≤–Ω–∏–≤–∞–µ–º 5 –∏ 1 ‚Üí –º–µ–Ω—è–µ–º ‚Üí `[3, 4, 1, 5]`  

–ü—Ä–æ—à–ª–∏—Å—å –ø–æ –º–∞—Å—Å–∏–≤—É –ø–µ—Ä–≤—ã–π —Ä–∞–∑. –¢–µ–ø–µ—Ä—å —Å–∞–º–æ–µ –±–æ–ª—å—à–æ–µ —á–∏—Å–ª–æ —É –Ω–∞—Å —Å–∞–º–æ–µ –ø—Ä–∞–≤–æ–µ. 
–ò–¥–µ–º –µ—â–µ —Ä–∞–∑ —Å –Ω–∞—á–∞–ª–∞ –º–∞—Å—Å–∏–≤–∞:

1. –°—Ä–∞–≤–Ω–∏–≤–∞–µ–º 3 –∏ 4 ‚Üí –≤—Å–µ –≤ –ø–æ—Ä—è–¥–∫–µ ‚Üí –∏–¥–µ–º –¥–∞–ª—å—à–µ

–ü—Ä–æ—à–ª–∏—Å—å –ø–æ –º–∞—Å—Å–∏–≤—É –≤—Ç–æ—Ä–æ–π —Ä–∞–∑.

1. –°—Ä–∞–≤–Ω–∏–≤–∞–µ–º 3 –∏ 1 ‚Üí –º–µ–Ω—è–µ–º ‚Üí `[1, 3, 4, 5]` ‚Üí –º–∞—Å—Å–∏–≤ –æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ωüëèüèªüéâ

**–ö–∞–∫ –ø—Ä–æ–≤–µ—Ä–∏—Ç—å, —á—Ç–æ –º–∞—Å—Å–∏–≤ –æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω?** 
–ù–∞ —Ç–∞–∫–æ–π —Å–ª—É—á–∞–π –∑–∞–≤–æ–¥–∏–º `bool-–ø–µ—Ä–µ–º–µ–Ω–Ω—É—é` (–∫–∞–∫ –ø—Ä–∞–≤–∏–ª–æ, –µ–µ –Ω–∞–∑—ã–≤–∞—é—Ç `flag`, –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é –æ–±—ã—á–Ω–æ `false`). –ö–æ–≥–¥–∞ –±—É–¥–µ–º —Å–æ–≤–µ—Ä—à–∞—Ç—å –ø–µ—Ä–µ—Å—Ç–∞–Ω–æ–≤–∫—É —ç–ª–µ–º–µ–Ω—Ç–æ–≤, –±—É–¥–µ–º –¥–µ–ª–∞—Ç—å `flag = true`.

–ü–æ—Å–ª–µ –ø—Ä–æ—Ö–æ–¥–∞ –ø–æ –º–∞—Å—Å–∏–≤—É `flag = false` ‚Üí  –ø–µ—Ä–µ—Å—Ç–∞–Ω–æ–≤–æ–∫ –Ω–µ –±—ã–ª–æ ‚Üí –º–∞—Å—Å–∏–≤ —É–∂–µ –æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω, –º–æ–∂–Ω–æ **–∑–∞–≤–µ—Ä—à–∏—Ç—å —Ä–∞–±–æ—Ç—É –∞–ª–≥–æ—Ä–∏—Ç–º–∞**.

void bubbleSort(std::vector<int>& a) {
    int n = a.size();
    for (int i = 0; i < n - 1; ++i) {
        bool flag = false;
        for (int j = 0; j < n - i - 1; ++j) {
            if (a[j] > a[j + 1]) {
                std::swap(a[j], a[j + 1]);
                flag = true;
            }
        }
        if (!flag) break; // –µ—Å–ª–∏ –Ω–∏ –æ–¥–Ω–æ–≥–æ —Å–≤–∞–ø–∞ –Ω–µ –±—ã–ª–æ -> –º–∞—Å—Å–∏–≤ –æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω
    }
}
```

void insertionSort(std::vector<int>& a) {
    int n = a.size();
    for (int i = 1; i < n; ++i) {
        int temp = a[i];
        int j = i - 1;

        while (j >= 0 && a[j] > temp) {
            a[j + 1] = a[j];
            j--;
        }
        a[j + 1] = temp;
    }
}
```

}

void binaryInsertionSort(std::vector<int>& a) {

    }
}
